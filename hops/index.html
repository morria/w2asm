Commands

.help
Get information on the bot (sends you here). Also: .info, .?
.whoami
Node info as Hops sees it
.ping
Sends a short ack
.hello
Say hello! Also: .👋🏼
.message <name> <message>
Send a message to a user by short or long name
.mail
Get your most recent 5 messages
.post <message>
Add a message to the global BBS
.bbs
List the most recent 5 BBS messages
